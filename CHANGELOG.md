# Changelog

Этот файл хранит только краткое актуальное состояние разработки. Подробную историю изменений следует смотреть в Git.

## Текущее Состояние

### Добавлено

- Desktop UI `ui.py` с адаптивным стартовым размером окна.
- Мастер первого запуска для пользователя без навыков программирования.
- Чат с ассистентом и отображением голосовых команд.
- Переключатель голосового считывания через `data/runtime/control.json`.
- История голосовых и текстовых команд в SQLite.
- Заметки, таймеры и напоминания.
- Подтверждение опасных действий.
- Профили ассистента: язык, голос, режим, приватность.
- Rutube-поиск вместо YouTube.
- Интеграции ЯндексGPT, погоды и новостей через UI.
- Live log, health snapshot, crash summary и support-отчёт с маскировкой секретов.
- Диагностика `diagnose.py` без запуска микрофонного прослушивания.
- Скрипты `install.bat`, `run_settings.bat`, `run_assistant.bat`, `run_diagnostics.bat`.
- Русскоязычная документация для пользователя и разработки.

### Изменено

- Целевой runtime: Python 3.11.
- Конфигурация объединяется с дефолтами из `src/core/config.py`.
- UI по умолчанию открывается в большем адаптивном размере.
- История и основные пользовательские списки в UI показывают русские подписи.
- Ошибки ЯндексGPT стали понятнее, включая несовпадение ID каталога.

### Пока Не Входит

- Финальная сборка `.exe`.
- Инсталлятор.
- Подписанные release-артефакты.
