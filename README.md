# Voice Assistant Cry

Voice Assistant Cry - Windows-ориентированный голосовой ассистент на Python с офлайн/онлайн распознаванием, озвучиванием, чат-интерфейсом, историей команд, заметками, напоминаниями, подтверждением опасных действий и диагностикой.

Обычному пользователю не нужно редактировать код или YAML вручную: основные настройки доступны в desktop UI.
При первом запуске можно включить автозапуск вместе с Windows, сворачивание в системный трей и автоматический старт голосового ассистента.

## Быстрый Старт

1. Установите Python 3.11.
2. Запустите установку зависимостей:

```bat
install.bat
```

3. Откройте настройки:

```bat
run_settings.bat
```

4. Пройдите мастер первого запуска.
5. Запустите ассистента:

```bat
run_assistant.bat
```

Диагностика без запуска микрофона:

```bat
run_diagnostics.bat
```

## Основные Возможности

- Голосовые команды через Vosk офлайн и SpeechRecognition онлайн.
- Чат с ассистентом, работающий через тот же движок команд.
- Озвучивание через Silero или `pyttsx3`.
- ЯндексGPT для неизвестных вопросов при настроенных ключах.
- Погода, новости, Rutube-поиск, системные команды, приложения.
- Заметки, таймеры, напоминания и история команд в SQLite.
- Настраиваемые подтверждения опасных команд.
- Профили ассистента, экспорт/импорт настроек и support-отчёт.

## Ограничения Перед Финальной Сборкой

- Целевая версия Python: 3.11.
- Проект рассчитан на Windows.
- Офлайн-распознавание требует Vosk-модели.
- ЯндексGPT, погода и новости требуют интернет и API-ключи.
- Текущий способ сборки: PyInstaller `onedir` + Inno Setup. Подробности: [docs/PACKAGING.md](docs/PACKAGING.md).
