# Руководство Пользователя

Этот гайд рассчитан на Windows-пользователя, который хочет пользоваться ассистентом без программирования и ручного редактирования файлов.

## Установка

1. Установите Python 3.11.
2. Откройте папку проекта.
3. Запустите:

```bat
install.bat
```

Скрипт создаст `.venv` и установит зависимости из `requirements.txt`.

## Первый Запуск

Откройте настройки:

```bat
run_settings.bat
```

При первом запуске откроется мастер настройки. В нём выбираются:

- язык ассистента;
- режим работы: офлайн, онлайн или гибридный;
- слово активации;
- город для погоды;
- движок озвучивания;
- запуск вместе с Windows и поведение системного трея;
- пути к основным приложениям.

Мастер также показывает чеклист готовности: Python, зависимости, микрофон, Vosk-модели, TTS, база SQLite, журналы, интеграции и пути приложений.

Предупреждения не всегда блокируют запуск. Ошибки лучше исправить до нажатия `Сохранить и завершить`.

## Запуск Ассистента

```bat
run_assistant.bat
```

Если первичная настройка не завершена, ассистент откроет UI и не начнёт слушать микрофон.

## Автозапуск И Трей

В мастере первого запуска и разделе `Ассистент` можно включить:

- `Запускать Cry вместе с Windows`;
- `При автозапуске открывать приложение в трее`;
- `Сворачивать окно в трей вместо закрытия`;
- `Запускать голосового ассистента при открытии приложения`.

Это пользовательский автозапуск, а не Windows Service. Такой режим нужен, чтобы приложение имело доступ к микрофону, UI и значку в системном трее.

Если включено сворачивание в трей, крестик скрывает окно, но приложение продолжает работать. В меню значка трея доступны команды `Показать окно`, `Скрыть в трей`, `Запустить/остановить ассистента` и `Выход`.

Проверьте их сначала в чате:

| Команда | Что проверяет |
| --- | --- |
| `что ты умеешь` | Список возможностей. |
| `примеры команд` | Примеры фраз. |
| `проверь ассистента` | Диагностика из навыка. |
| `последние команды` | История команд. |

## Чат

Чат принимает те же команды, что и голосовой режим. Если команда не совпала с локальным навыком и включён ЯндексGPT, вопрос отправляется в ЯндексGPT.

Кнопка `Голос` включает или отключает считывание микрофонных команд для запущенного ассистента. Текстовый чат продолжает работать всегда.

## Голосовые Команды

Сначала произнесите слово активации, затем команду:

```text
край открой браузер
край найди на рутубе новости технологий
край сколько времени
край последние команды
край поставь таймер на пять минут
```

Слова активации меняются в разделе `Ассистент`.

## Профили

Раздел `Профили` сохраняет набор персональных настроек:

- имя ассистента;
- язык;
- голос;
- слова активации;
- режим работы;
- настройки приватности.

Профили не сохраняют API-ключи и пути приложений. Это сделано специально, чтобы профиль можно было переносить без секретов.

## Интеграции

### ЯндексGPT

1. Откройте `Интеграции`.
2. Включите `Отвечать через ИИ на неизвестные команды`.
3. Заполните `API-ключ ЯндексGPT`.
4. Заполните `ID каталога Яндекс Облака`.
5. Нажмите `Сохранить`.
6. Нажмите `Обновить контекст` в чате или перезапустите окно настроек.

Если API-ключ принадлежит сервисному аккаунту из другого каталога, ЯндексGPT вернёт ошибку несовпадения ID каталога.

### Погода

1. Получите ключ OpenWeatherMap.
2. Заполните `API-ключ OpenWeatherMap`.
3. Укажите город по умолчанию.
4. Проверьте команду `какая погода`.

### Новости

1. Получите ключ NewsAPI.
2. Заполните `API-ключ NewsAPI`.
3. Проверьте команду `последние новости`.

## История

В разделе `История` доступны фильтры:

- источник: `Все`, `Голос`, `Чат`;
- статус: `Выполнено`, `Не распознано`, `Требуется подтверждение`, `Ответ ИИ`, `Ошибка`;
- текстовый поиск по фразе, действию, шаблону и ответу.

`Копировать детали` удобно использовать, если нужно передать проблему распознавания.

## Команды И Подтверждения

В разделе `Команды` можно:

- искать встроенные команды;
- добавлять свои фразы к существующим действиям;
- включать или выключать подтверждение для выбранной команды;
- проверять фразу без выполнения действия.

Опасные команды требуют подтверждения, например выключение ПК, очистка истории/журналов/заметок или закрытие приложений.

Выключение и перезагрузка запускаются не мгновенно: после подтверждения Windows получает задачу с задержкой. Пока время не истекло, можно сказать `отмени выключение` или `отмени перезагрузку`.

Для подтверждения скажите:

```text
подтверждаю
```

Для отмены:

```text
отмена
```

## Заметки И Напоминания

Заметки и напоминания хранятся в SQLite-базе `data/assistant.sqlite3`.

Поддерживаемые форматы времени в UI:

```text
10m
2h
2026-06-10 18:30
18:30
```

Голосом и в чате также можно использовать фразы вроде:

```text
через полчаса
завтра в 18:30
сегодня в 18:30
```

## Диагностика И Support

Раздел `Диагностика` умеет:

- проверить конфигурацию;
- запустить `diagnose.py`;
- показать последние строки журнала;
- показать health snapshot;
- показать crash summary;
- экспортировать отчёт поддержки;
- проверить заполнение секретов;
- очистить API-ключи;
- создать и восстановить резервную копию `config.yaml`.

Support-отчёт по умолчанию скрывает API-ключи. История команд включается в отчёт только если это разрешено в `Ассистент -> Приватность и support`.

## Журналы

Основной журнал:

```text
data/logs/assistant.log
```

Вывод процесса, запущенного из UI:

```text
data/logs/assistant_stdout.log
```

Для просмотра используйте кнопку `Живой журнал`.

## Рекомендуемый Сценарий

1. Запустите `install.bat`.
2. Запустите `run_settings.bat`.
3. Пройдите мастер первого запуска.
4. Проверьте микрофон и голос.
5. Проверьте чат командой `что ты умеешь`.
6. Настройте интеграции при необходимости.
7. Запустите `run_assistant.bat`.
8. Используйте голосовые команды со словом активации.
